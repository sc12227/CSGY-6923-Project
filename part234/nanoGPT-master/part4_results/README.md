`conditional_abc` folder contains all the abc results of the conditional generation.

`unconditional_abc` folder contains all the abc results of the unconditional generation.

`part4_ans.txt` contains the results for each epoch and the final test results.

`batch_abc_to_midi_and_stats.py` contains the script for converting the generated ABC files into audio files.

`midi_out_cond` folder folder contains all the mid results of the conditional generation.

`midi_out_uncond` folder folder contains all the mid results of the unconditional generation.
